# Сайдлоад Orbit

## Что скачивать

В каждом релизе есть файл вида `Orbit-1.0-buildXXX-Feather.ipa`. Это device-сборка `arm64` без подписи разработчика — её нужно подписать на своём устройстве через Feather, AltStore или Sideloadly.

В пакет намеренно входит только одно вложенное расширение: `PacketTunnel.appex`. Оно необходимо для VPN. Виджет остаётся в исходниках, но не упаковывается в Feather-вариант, чтобы избежать проблем установки после повторной подписи.

## Установка

1. Откройте последнюю страницу [Releases](https://github.com/Lendic42/Orbit/releases/latest) и скачайте IPA.
2. Импортируйте файл в Feather, AltStore или Sideloadly.
3. Подпишите сборку своим сертификатом и завершите установку.
4. При первом запуске разрешите Orbit добавить VPN-конфигурацию.

После изменения Bundle ID приложение устанавливается как новый экземпляр. Если старый Orbit уже был на телефоне, не нужно удалять его заранее: сначала проверьте новую сборку и импорт конфигурации.

## Если приложение не устанавливается

- Удалите только неудачную копию Orbit из sideload-инструмента и повторите подпись IPA.
- Убедитесь, что в пакете не остаётся старый вариант с виджетом: используйте именно файл с суффиксом `-Feather.ipa`.
- Проверьте, что сертификат разработчика не истёк и что sideload-инструмент подписал и приложение, и PacketTunnel.
- После установки откройте приложение хотя бы один раз до попытки создать VPN.

## Первый импорт

На главном экране есть отдельные кнопки для HTTPS-подписки, `qwdtt://`, `wdtt://` и QR-кода. Копируйте ссылку целиком, без лишнего текста вокруг неё. После успешного импорта qWDTT-профиль активируется и Orbit предложит подключение.
